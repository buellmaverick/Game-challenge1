🎱 The Pool
Every physical game you own enters the pool. Previously beaten games get no exemption — clean slate, everything in. New purchases added immediately upon returning home.
🎲 Drawing a Game
When you need a new game you randomly draw from the pool in that moment. No predetermined order — every draw is fresh. Any game can come up at any time.
🎮 Playing a Game
Play whatever was drawn until beaten. Main credits rolling = beaten. Post-credits optional. One game at a time — no switching.
🏆 Beating & Removing
Beaten game is permanently removed from the pool. No exceptions.
🚪 Quitting a Game
Must play a minimum number of hours first. Must provide honest good-faith reasoning. No quitting just because it's hard. Quit game permanently removed.
⚠️ Unbeatable Games
Sports, party games, games without endings get custom completion rules decided before play begins.
🛒 Buying Rules
Every 2 weeks buy one physical used game from a brick and mortar store. Spin the wheel to pick the console. Your choice of game. Added to pool immediately.
⭐ New Release Unlock
Every 5 games beaten unlocks one new release reward. Current year, physical, brick and mortar. Skips pool — played immediately.
🔧 Hardware Rules
Real hardware only. If it breaks, repair or replace before continuing. Save corruption = start over, no exceptions. Modded hardware OK for physical media only.
✅ What Counts
✓ Physical retail games on real hardware
✓ GameCube discs on Wii
✓ Physical Vita carts on modded Vita
✓ Collection discs — all games played individually
❌ What Doesn't Count
✗ Digital purchases
✗ Emulation
✗ Steam / PC
✗ Mini consoles
✗ Multi-carts
✗ Games via mod/HDD/SD adapter
