🎱 The Pool
Every physical game you own enters the pool. Previously beaten games get no exemption — clean slate, everything in. New purchases are added immediately. The pool grows and shrinks over time.
🎲 Drawing a Game
When you need a new game you randomly draw from the pool in that moment. No predetermined order — every draw is fresh. Any game currently in the pool can come up at any time.
🎮 Playing a Game
Play whatever was drawn until it is beaten. Main credits rolling = beaten. Post credits content is optional. One game at a time — no switching until current game is beaten or legitimately quit.
🏆 Beating & Removing
Beaten game is permanently removed from the pool. Exception: if a disc or cart contains multiple games, they are played in release order. Beat one → removed, next title re-enters the pool until all titles are completed.
🚪 Quitting a Game
Must play a minimum number of hours before quitting. Must provide honest good faith reasoning. No quitting just because it's hard. Quit game is permanently removed from the pool.
⚠ Unbeatable Games
Sports, party games and games without endings get custom completion rules decided before the game begins. Must be agreed and logged before playing starts.
🛒 Buying Rules
Every 2 weeks purchase one physical used game from a brick and mortar store. Randomly spin to determine which console you are buying for. Your choice of which game to buy. Game is added to the pool immediately.
⭐ New Release Unlock
Every 5 games beaten unlocks one new release reward. Must be current year, physical, bought from a brick and mortar store. Reward game skips the pool and is played next immediately.
🔧 Hardware Rules
Real hardware only — no mini consoles, no emulation. If hardware breaks it must be repaired or replaced before continuing. Save corruption = start over, no exceptions. Modded hardware permitted for physical media only.
✅ What Counts
✓ Physical retail games on real hardware
✓ Collection discs — all titles played in order
✓ Physical Vita carts on modded Vita
✓ GameCube discs played on Wii
❌ What Doesn't Count
✗ Digital purchases of any kind
✗ Emulation of any kind
✗ Steam or PC gaming
✗ Mini consoles
✗ Multi-carts
✗ Games loaded via mod, HDD, or SD adapter
